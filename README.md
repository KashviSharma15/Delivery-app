# Delivery-app

A modern and responsive **Food Delivery Application** that enables users to browse restaurants, explore menus, place orders, and track deliveries seamlessly. The application provides an intuitive user experience for customers while supporting efficient order management.

---

## 🚀 Features

### 👤 User Features
- User Registration & Login
- Secure Authentication
- Browse Restaurants
- Search & Filter Restaurants
- View Food Menu
- Add/Remove Items from Cart
- Update Item Quantity
- Place Orders
- Order History
- Real-time Order Status
- Responsive UI

### 🛒 Cart Management
- Add items to cart
- Remove items from cart
- Update quantities
- Automatic total price calculation

### 📦 Order Management
- Place new orders
- View previous orders
- Track order status
- Order summary

### 🔍 Search & Filter
- Search restaurants by name
- Filter menu items
- Category-based browsing

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/KashviSharma15/Delivery-app.git
```

### Navigate to the project

```bash
cd Delivery-app
```

### Install dependencies

```bash
npm install
```

### Start the application

```bash
npm start
```
